 ---------------------- HELP ---------------------------------

 Для работы с Netti-Nio Command Line File Manager используйте следующие команды:
 *UPLOAD     - Отправка файлов на сервер;
 *DOWNLOAD   - Скачивание файлов с сервера;
 *SHOW       - Просмотр файлов на сервере для каждого пользователя;
 *DELETE     - Удаление файлов на сервере;
 *RENAME     - Переименование файлов на сервере;
 *HELP       - Вывод данной справки
 *HELLO      - Вывод информации о программе.
 *EXIT       - Завершение программы

 Знак "*" вначале каждой команды обязателен!
 Команды могут быть введены в любом регистре.
-------------------------------------------------------------



 *UPLOAD     - Отправка файлов на сервер;
 Для отправки файла введите имя файла, находящегося в вашей папке на компьютере.
 В случае, если на сервере файл с таким именем уже существует, то он будет переименован путем добавления к имени файла "old" к имени файла.
 В случае, если переименвоание снова порождает конфликт имен, файлы будут переименовываться до тех пор, пока конфликт не будет исчерпан.

 *DOWNLOAD   - Скачивание файлов с сервера;
Для отправки файла введите имя файла, находящегося в вашей папке на компьютере.

 *SHOW       - Просмотр файлов на сервере для каждого пользователя;

 *DELETE     - Удаление файлов локально/на сервере;
 Для удаления файла введите его имя  или /n - где n-номер файла, который вы получите при использовании команды *show


 *RENAME     - Переименование файлов на сервере;
 Для переименования файла введите:
   имя файла, который хотите переименовать, или /n - где n - номер файла на сервере ( который вы получите при использовании команды *show)
   затем введите -> новое имя файла, после этого нажмите ввод.
 Примеры команд:
    /12 -> newText      - переименует файл под номером 12 на "newText"
    text -> newText     - переименует файл с именем "text" на "newText"

 *HELP       - Вывод данной справки

 *HELLO      - Вывод информации о программе.

 *EXIT       - Завершение программы
