 Для работы с Netti-Nio Command Line File Manager используйте следующие команды:
 *UPLOAD     - Отправка файлов на сервер;
 *DOWNLOAD   - Скачивание файлов с сервера; *
 *SHOW       - Просмотр файлов на сервере для каждого пользователя;
 *DELETE     - Удаление файлов локально/на сервере;
 *RENAME     - Переименование файлов локально/на сервере;
 *HELP       - Вывод данной справки
 *HELLO      - Вывод информации о программе.
 *EXIT       - Завершение программы

 Знак "*" вначале каждой команды обязателен!
 Команды могут быть введены в любом регистре.
